# 成本对比

## 方案A：免费方案

**工具栈**：
- 剧本：GPT-3.5-turbo ($0.002/1K tokens)
- 配音：Edge TTS (免费)
- 视频：SadTalker (本地免费)

**成本**：
- 100集剧本：约 $2-5
- 配音：$0
- 视频：$0 (仅电费)
- **总计：$2-5**

## 方案B：付费方案

**工具栈**：
- 剧本：GPT-4 ($0.03/1K tokens)
- 配音：Azure TTS ($16/百万字符)
- 视频：SadTalker (本地)

**成本**：
- 100集剧本：约 $20-30
- 配音：约 $5-10
- 视频：$0
- **总计：$25-40**

## 推荐

测试期用免费方案，有爆款后升级付费方案提升质量。
