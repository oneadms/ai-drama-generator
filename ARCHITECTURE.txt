┌─────────────────────────────────────────────────────────────────┐
│                         AI 短剧生成系统                          │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                          配置层 (Config)                         │
├─────────────────────────────────────────────────────────────────┤

│                        工厂层 (Factory)                          │

│  create_voice_generator()                                       │
│  create_video_generator()                                       │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                        抽象层 (Traits)                           │
├─────────────────────────────────────────────────────────────────┤
│  trait ScriptGenerator  →  generate(episode, context)          │
│  trait VoiceGenerator   →  generate(text, output)              │
│  trait VideoGenerator   →  generate(audio, output)             │
│  trait Uploader         →  upload(video, metadata)             │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                      实现层 (Generators)                         │
├──────────────────┬──────────────────┬──────────────────────────┤
│  ScriptGenerator │  VoiceGenerator  │  VideoGenerator          │
├──────────────────┼──────────────────┼──────────────────────────┤

└──────────────────┴──────────────────┴──────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                       流水线 (Pipeline)                          │
├─────────────────────────────────────────────────────────────────┤
│  for episode in 1..total:                                       │

│    3. 生成视频  →  video_gen.generate()                         │

└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐

└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
